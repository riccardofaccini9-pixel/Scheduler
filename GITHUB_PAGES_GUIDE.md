# Guida alla pubblicazione su GitHub Pages

Segui questa guida passo-passo per pubblicare **ChoreFlow** online come sito web pubblico o privato accessibile da smartphone e computer.

---

## 1. Crea un repository su GitHub
1. Accedi a [GitHub](https://github.com/) (se non hai un account, crealo gratuitamente).
2. Fai clic sul pulsante **New** (Nuovo repository) in alto a sinistra.
3. Dai un nome al repository, ad esempio: `choreflow`.
4. Scegli se renderlo **Public** (consigliato per l'attivazione gratuita di GitHub Pages).
   > [!NOTE]
   > Se crei un repository *Private*, l'hosting su GitHub Pages richiede un piano a pagamento (GitHub Pro).
5. Lascia deselezionate le opzioni di inizializzazione (README, .gitignore, licenza).
6. Fai clic su **Create repository**.

---

## 2. Carica i file di ChoreFlow
Puoi caricare i file direttamente dal browser web:
1. Nella pagina del repository appena creato, fai clic sul link **uploading an existing file** (carica un file esistente).
2. Trascina i seguenti file e cartelle dalla cartella locale di ChoreFlow all'area di caricamento di GitHub:
   - `index.html`
   - `style.css`
   - `app.js`
3. Attendi il caricamento di tutti i file.
4. In fondo alla pagina, digita un messaggio per il caricamento (es. `Inizializzazione ChoreFlow`) e fai clic su **Commit changes**.

---

## 3. Attiva GitHub Pages
1. Vai alla scheda **Settings** (Impostazioni) in alto nel tuo repository GitHub.
2. Nel menu laterale a sinistra, fai clic sulla sezione **Pages** (sotto "Code and automation").
3. Sotto **Build and deployment**:
   - Per **Source**, seleziona `Deploy from a branch`.
   - Per **Branch**, seleziona `main` (o `master`) e seleziona `/ (root)` come directory.
4. Fai clic su **Save**.

---

## 4. Accedi al sito web
1. Attendi circa 1-2 minuti.
2. Aggiorna la pagina delle impostazioni di GitHub Pages: vedrai un box in alto con scritto:  
   `Your site is live at https://<tuo-username>.github.io/choreflow/`
3. Condividi questo link con i membri della casa per consentire loro di accedere come Cadetti tramite smartphone o PC!
